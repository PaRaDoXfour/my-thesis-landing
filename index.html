Сумський державний університет

Інформаційна система інтелектуального керування бібліотечним каталогом з підтримкою електронних та аналогових носіїв інформації
Information system for intelligent library catalog management with support for electronic and analog media

Студент: Костян Олег, група ІН-22.1

Короткий опис: Сучасне програмне рішення для трансформації традиційних бібліотек у мультимедійні центри шляхом управління гетерогенними колекціями: цифровими, аналоговими та рідкісними ресурсами.

Ключові слова: Інформаційна система, Управління бібліотекою, Java, Мультимедійний каталог, ООП

Про проєкт

Актуальність

Сучасні бібліотеки трансформуються з класичних книгосховищ у мультимедійні центри. Традиційні підходи до обліку не здатні врахувати специфічні атрибути нових медіа-ресурсів, таких як формат файлу, тривалість аудіо чи унікальна вартість антикваріату. Робота спрямована на подолання конфлікту між уніфікацією процесів та унікальністю кожного типу ресурсу.

Мета та об'єкт дослідження

Мета: Проєктування та розробка системи на Java з використанням принципів ООП для автоматизації обліку різних типів носіїв.

Об'єкт дослідження: Процес управління бібліотечними фондами, що містять різнотипні медіа-ресурси.

Завдання:

  • Аналіз атрибутів предметної області (ISBN, формати, історична цінність).
  • Проєктування архітектури (успадкування, поліморфізм).
  • Реалізація функціоналу (CRUD, пошук, серіалізація даних).
  • Розробка інтерфейсу користувача.

Новизна та цінність

Практичне застосування

Новизна: Гібридна модель даних та практичне застосування для шкіл і бібліотек.
